Algorithm Steps (short)
  1. Eliminate obvious composites: n < 2, even numbers, or divisible by small primes.
  2. Run Fermat tests for several bases a: check a^(n-1) mod n = 1.
  3. If any test fails, n is composite; if all pass, n is probably prime.
